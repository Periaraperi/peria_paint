# peria_paint

*This project is a work in progress, however it is already usable for simple note taking or drawings.* 
 
## Why create a digital painting program?

I really liked Windows 7 and 10 version of MS Paint. I used it for note taking, whiteboard, paper alternative and for silly joke drawings. I really like its simplicity. I don't like Windows 11 version and furthermore it is not available on Linux. There are many great digital drawing software such as Krita or GIMP, however they are pretty complex and an overkill for my needs. I also didn't enjoy other alternatives such as MyPaint, Pinta, KolourPaint. So I decided to write my own for my day to day usage. It is also fun and great learning experience.
